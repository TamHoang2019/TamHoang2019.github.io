Font local

Các font có sẵn trên máy tính, hoặc được download từ trên mạng về máy tính.

- Bước 1: copy file font(thường là có đuôi .ttf hoặc .otf) vào thư mục fonts của website.
- Bước 2: Trong file .css đặt tên font, và khai báo đường dẫn tới file font như sau
@font-face{
src: url(đường dẫn tới tên font);
font-family: tên-font-tự-đặt; }
- Bước 3: Mỗi khi cần sử dụng font này cho một thẻ nào đó trong file .css, thì gọi tên font như sau
h1{
font-family: tên-font-đã-đặt; }

Font google

Là các font được lấy link trực tiếp từ google, không cần tải về máy tính.

- Bước 1: vào trang fonts.google.com .
- Bước 2: Chọn một hoặc nhiều font ưa thích
- Bước 3: Copy đường link, rồi paste vào file .html, ví dụ
<link href="https://fonts.googleapis.com/css?family=Lato&display=swap" rel="stylesheet">
- Bước 4: Mỗi khi cần sử dụng font này cho một thẻ nào đó trong file .css, thì gọi tên font như sau
h1{ font-family: tên-font-do-google-đặt; }
ví dụ
font-family: 'Lato', sans-serif;
font-family: 'Roboto', sans-serif;

Fontawesome

Là tạo icon cho website không cần hình ảnh, mà dùng fontawesome, sử dụng giống như là font bình thường, nghĩa là có thể đổi màu sắc, kích thước, vị trí... của icon một cách dễ dàng.

- Bước 1: vào trang fontawesome.com .
- Bước 2: Download và giải nén, lấy folder fonts, và file css font-awesome.css bỏ vào folder website của mình, folder fonts đặt ngang cấp với file index.html.
- Bước 3: Trong file index.html trong thẻ head khai báo sử dụng fontawesome như sau:
<link rel="stylesheet" href="css/font-awesome.min.css">

Có 2 cách dùng fontawesome

1. Dùng trực tiếp trong file .html
Dùng bất kỳ tên thẻ nào cũng được, kèm theo 2 class .fa.fa-tên-của-icon-do-fontawesome-đặt.
ví dụ <div class="fa fa-heart">
Khi ra trình duyệt sẽ thấy kết quả là hình trái tim.
Chú ý: fa fa- là quy định của fontawesome. Tuỳ theo phiên bản sẽ có khác biệt, nên vào từng icon xem hướng dẫn. Còn phiên bản 4 thì quy tắc chung là .fa.fa- là được.
2. Dùng gián tiếp trong file .css thông qua pseudo-element ::before, hoặc ::after. Khi này, chúng ta phải tìm được mã code của icon cần dùng, để paste vào thuộc tính content của pseudo-element đó.
ví dụ
li::before{
content:"\f0c8";
font-family: FontAwesome; //đối với fontawesome phiên bản 4.
font-family: "Font Awesome 5 Free";; //đối với fontawesome phiên bản 5. }

Overflow

Là thuộc tính giúp chúng ta quy định cách xử lý phần nội dung bị dư ra khỏi width và height của thẻ chứa nó.

Vấn đề:

Thường khi chúng ta float left hoặc right cho một thẻ nào đó, thì nội dung của thẻ bên cạnh sẽ tràn qua phần còn trống của thẻ đó.

Để khắc phục, muốn xếp gọn gàng 2 bên nội dung không lấn chiếm lẫn nhau, ta dùng thuộc tính overflow:hidden;
.right{
overflow: hidden;
}

Transition

Là thuộc tính quản lý thời gian thực hiện chuyển động.

- Nếu transition không hoạt động, nghĩa là bạn đã thiếu định nghĩa ra một thuộc tính css tạo ra hiệu ứng. (Ví dụ hiệu ứng di chuyển lên xuống, trái phải, ẩn, hiện...)
- Ví dụ:
Bạn muốn transition cho thẻ a, dịch chuyển vị trí top xuống 50px, là 0.4s.
a:hover {
top: 50px;
transition: 0.4s;
}
Nhưng nếu ban đầu, bạn không định nghĩa vị trí top cho thẻ a, thì khi transition nó sẽ ko thực hiện.
Nghĩa là bắt buộc phải có luôn css như sau:
a{
top:0;
}
Lúc này, trình duyệt mới hiểu là bạn muốn thay đổi từ top là 0 đến 50px trong thời gian 0.4s
