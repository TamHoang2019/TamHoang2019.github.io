Font local

Các font có sẵn trên máy tính, hoặc được download từ trên mạng về máy tính.

- Bước 1: copy file font(thường là có đuôi .ttf hoặc .otf) vào thư mục fonts của website.
- Bước 2: Trong file .css đặt tên font, và khai báo đường dẫn tới file font như sau
@font-face{
src: url(đường dẫn tới tên font);
font-family: tên-font-tự-đặt; }
- Bước 3: Mỗi khi cần sử dụng font này cho một thẻ nào đó trong file .css, thì gọi tên font như sau
h1{
font-family: tên-font-đã-đặt; }

Font google

Là các font được lấy link trực tiếp từ google, không cần tải về máy tính.

- Bước 1: vào trang fonts.google.com .
- Bước 2: Chọn một hoặc nhiều font ưa thích
- Bước 3: Copy đường link, rồi paste vào file .html, ví dụ
<link href="https://fonts.googleapis.com/css?family=Lato&display=swap" rel="stylesheet">
- Bước 4: Mỗi khi cần sử dụng font này cho một thẻ nào đó trong file .css, thì gọi tên font như sau
h1{ font-family: tên-font-do-google-đặt; }
ví dụ
font-family: 'Lato', sans-serif;
font-family: 'Roboto', sans-serif;
